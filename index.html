Septemder 01SUN, 2024

Developing a Satisfaction Survey Website for Asia Technology College

During my studies at Asia Technology College, I had the opportunity to develop a satisfaction survey website. The website was designed to evaluate three important student projects: the "Multi-Purpose Vending Machine," the "Cricket Protein Cookies," and the "Laser Staff Walking Stick." Each of these projects showcased different skills and innovation from the students. For the development of this website, I used HTML, CSS, and JavaScript as the main technologies to ensure ease of use and efficient functionality. The website was designed with simplicity in mind, making it user-friendly and accessible. It allows users to easily evaluate the projects and provide their feedback quickly and effectively. The website features pages where users can select the project they want to review. These include the "Multi-Purpose Vending Machine," designed for user convenience; the "Cricket Protein Cookies," a nutritional product made from crickets; and the "Laser Staff Walking Stick," designed to assist individuals with visual impairments. Users can rate each project based on their satisfaction and submit their opinions or suggestions through Google Sheets for data collection and further analysis. The website is now available online at https://asia-lb.web.app. Developing this website not only enhanced my technical skills but also contributed to the students' ability to present their work to the public and receive valuable feedback for improvement in the future.

Listing Directory Website Project for Free Trial Software Distribution

I had the opportunity to develop a Listing Directory website project with the primary goal of collecting and distributing free trial software. This platform allows users to browse and download various trial software to try out. The website is designed to make it easy for anyone interested in exploring new software to find and access the tools they need. For this project, I used HTML, CSS, and JavaScript to build the website, focusing on user-friendly design and efficient data management. Users can browse through software categorized by different types, such as design software, development tools, data management applications, and more. Additionally, a search feature is provided to help users quickly find the software they're looking for. Each page on the website provides detailed information about each software, including descriptions, features, downloadable trial versions, and installation instructions. Users can freely download and install the software, and they also have the option to provide feedback or suggestions for future improvements. Creating this project has not only enhanced my knowledge of website design and data management, but also improved my programming skills, especially in building a system that serves a large number of users. It was a valuable experience in both technical development and user service management.
